# Use pytest in VSCODE

## Test with multiple input / output
Go to pytest folder
```bash
cd pytest_dr
```

Test file
```bash
pytest pytest_file.py
```

or just use command
```bash
pytest pytest_dr/pytest_file.py
```

## Simple test with 
```bash
pytest pytest_dr/assert_test.py
```

## Note
All the test function need to start with test
